## ✨ 구현된 기능

### 1. 프로필 등록
- 이름, 나이, 성별, 국가 입력
- 자기소개 및 관심사 작성
- 나이 제한: 40-70세
- 필수 항목 검증

### 2. 프로필 찾기
- 성별 필터링 (반대 성별만 표시)
- 국가 필터링 (다른 국가만 표시)
- 프로필 카드 형식으로 표시
- 매칭 요청 기능

### 3. 매칭 시스템
- 원클릭 매칭 요청
- 매칭 기록 저장
- 실시간 알림

### 4. 통계 대시보드
- 전체 회원 수
- 매칭 성공 수
- 성별 분포 (남성/여성)
- 연령대별 분포 (40대/50대/60대)

## 📡 API 엔드포인트

### POST /api/register
프로필 등록

**요청:**
```json
{
  "name": "홍길동",
  "age": 45,
  "gender": "male",
  "country": "한국",
  "about": "안녕하세요",
  "interests": "여행, 독서"
}
```

**응답:**
```json
{
  "success": true,
  "profile": {
    "id": 1,
    "name": "홍길동",
    "age": 45,
    "gender": "male",
    "country": "한국",
    "about": "안녕하세요",
    "interests": "여행, 독서",
    "createdAt": "2025-11-28T23:56:33.959Z"
  }
}
```

### GET /api/profiles
프로필 목록 조회

**쿼리 파라미터:**
- `gender`: 찾고자 하는 성별 (male/female)
- `country`: 제외할 국가

**응답:**
```json
{
  "profiles": [
    {
      "id": 1,
      "name": "홍길동",
      "age": 45,
      "gender": "male",
      "country": "한국",
      "about": "안녕하세요",
      "interests": "여행, 독서",
      "createdAt": "2025-11-28T23:56:33.959Z"
    }
  ]
}
```

### POST /api/match
매칭 요청

**요청:**
```json
{
  "fromId": 1,
  "toId": 2
}
```

**응답:**
```json
{
  "success": true,
  "message": "매칭 요청을 보냈습니다!"
}
```

### GET /api/stats
통계 조회

**응답:**
```json
{
  "totalProfiles": 7,
  "totalMatches": 2,
  "byGender": {
    "male": 3,
    "female": 4
  },
  "byAgeGroup": {
    "40s": 2,
    "50s": 3,
    "60s": 2
  }
}
```

## ⚠️ 알려진 제한사항

### 1. 데이터 휘발성
- 현재 In-Memory Map 사용
- 서버 재시작 시 모든 데이터 삭제됨
- **해결방안:** Cloudflare D1 또는 KV 사용 권장

### 2. 인증 시스템 없음
- 프로필 등록 시 인증 없음
- 누구나 매칭 요청 가능
- **해결방안:** JWT 또는 세션 기반 인증 필요

### 3. 이미지 업로드 미지원
- 텍스트 기반 프로필만 가능
- **해결방안:** Cloudflare R2 연동

### 4. 실시간 채팅 없음
- 매칭 후 연락 수단 없음
- **해결방안:** WebSocket 또는 외부 채팅 서비스 연동
