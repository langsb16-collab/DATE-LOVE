# Couple Gate - 국제 연애·결혼 매칭 플랫폼

## 📋 프로젝트 개요
- **이름**: Couple Gate
- **목표**: 40대·50대·60대 싱글, 돌싱을 위한 국경을 넘어 사랑을 연결하는 글로벌 국제 연애·결혼 매칭 플랫폼
- **주요 기능**:
  - 다국어 지원 (한국어, 영어, 중국어, 일본어, 베트남어, 아랍어)
  - AI 기반 프로필 매칭
  - 실시간 번역 및 화상통화
  - 문화 차이 코칭
  - 엄격한 인증 시스템 (3개 이상 SNS 인증)

## 🌐 URL
- **메인 페이지**: https://jtbit.me
- **관리자 페이지**: https://jtbit.me/admin
  - ID: `admin`
  - 비밀번호: `admin1234`
- **공지사항 페이지**: https://jtbit.me/notices

## 🤖 챗봇 기능
✅ **6개 언어 자동 응답 챗봇 구현 완료**
- **디자인**: 보라색 그라데이션 (#667eea → #764ba2) 테마
- **위치**: 하단 우측 로봇 아이콘 + AI 뱃지

  - 일본어 (ja) - 19개 FAQ
  - 베트남어 (vi) - 19개 FAQ
  - 아랍어 (ar) - 19개 FAQ
- **자동 동기화**: 사이트 언어 선택기와 챗봇 언어가 자동으로 동기화됨

## ✅ 완료된 기능
1. **회원 관리**
   - 프로필 등록, 조회, 수정, 삭제
   - 연령대, 성별, 국가별 필터링
   - 프로필 검색 기능

2. **매칭 관리**
   - 매칭 요청 및 상태 관리
   - 매칭 통계 및 분석

3. **관리자 기능**
   - 회원 관리 (조회, 수정, 삭제)
   - 매칭 관리
   - 통계 대시보드
     - 총 프로필 수
     - 매칭 수
     - 성별 비율
     - 연령대별 통계 (40대, 50대, 60대)
   - 공지사항 관리 (작성, 수정, 삭제)

4. **공지사항 시스템**
   - 중요 공지 표시 기능
   - 공지사항 목록 및 상세 페이지

5. **다국어 지원**
   - 6개 언어 완벽 지원
   - 언어별 UI 자동 전환

6. **자동응답 챗봇**
   - 6개 언어별 19개 FAQ
   - 첨부된 사진 디자인과 동일한 UI
   - 질문 클릭 방식 답변 제공

## 📊 데이터 구조
- **프로필 (Profile)**
  - id, name, age, gender, country, about, interests, createdAt
  
- **매칭 (Match)**
  - id, fromId, toId, status, createdAt
  
- **공지사항 (Notice)**
  - id, title, content, createdAt, important

- **권장**: Cloudflare D1 (SQLite) 또는 KV 사용

## 🛠️ 개발 및 배포 명령어

### 로컬 개발
```bash
# 의존성 설치
npm install

# 빌드 (esbuild로 번들링)
npm run build

# 로컬 미리보기
npm run preview
```

### 배포
```bash
# 빌드 + 배포
npm run deploy

# 또는 수동 배포
npm run build
npx wrangler pages deploy dist --project-name jtbit
```
